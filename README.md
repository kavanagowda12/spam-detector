# spam-detector
A machine learning–based spam detector that classifies messages as spam or not spam using NLP techniques. It preprocesses text, extracts features (TF-IDF/BoW), and applies models like Naive Bayes or SVM to deliver accurate predictions and efficient filtering of unwanted messages.
# Spam Detector 📧

A machine learning–based spam detector that classifies messages as spam or not spam using NLP techniques.

## Features
- Detects spam messages
- Uses machine learning models
- Fast and accurate predictions

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the app:
   python app.py

## Example
Input: "You won a free prize!"
Output: Spam
